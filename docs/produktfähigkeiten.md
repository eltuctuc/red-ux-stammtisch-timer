# Produktfähigkeiten

## Timer *(FEAT-2, seit 2026-03-29)*
Der Moderator wählt eine Timebox-Dauer per Preset (2, 5, 10, 15, 30 Min) oder individueller Eingabe und startet den Countdown mit einem Klick. Alle Teilnehmer der Session sehen denselben Timer synchron in Echtzeit, inklusive farblicher Zustandswechsel (running → warning bei ≤20% → expired) und einem akustischen Signal bei Ablauf. Der Moderator kann den Timer jederzeit pausieren, fortsetzen oder zurücksetzen; Teilnehmer sehen nur die Anzeige ohne Steuerungselemente.

## Session Management *(FEAT-1, seit 2026-03-28)*
Moderatoren können eine neue Workshop-Session starten und erhalten automatisch eine Moderatoren-URL (mit Secret-Token) sowie eine Teilnehmer-URL zum Teilen. Teilnehmer betreten die Session per direktem Link oder durch manuelle Eingabe der 4-stelligen Session-Nummer auf der Startseite. Sessions verfallen serverseitig nach 3 Stunden ohne Aktivität, und Moderatoren können ihre Session jederzeit per Moderatoren-URL wieder aufnehmen.
